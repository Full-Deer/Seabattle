# Морской бой (Sea Battle)

Консольная игра "Морской бой" на языке C с поддержкой русского языка и цветным интерфейсом.

## Описание

Классическая игра "Морской бой" между игроком и компьютером.
Игрок расставляет свои корабли на поле 10×10, после чего начинается поочерёдная стрельба.
Компьютер использует интеллектуальный алгоритм поиска кораблей.

### Особенности

- Игра против компьютера
- Цветной интерфейс с поддержкой ANSI
- Игра на русском языке
- Статистика игры (в доработке)
- Проработанный алгоритм стрельбы компьютера
- Возможность повторной игры

#### Сборка проекта

Очистка предыдущей сборки

	make clean

Сборка проекта

	make

Сборка с отладочной информацией

	make debug

Запуск собранной программы

	./program

Или после make

	make run

##### Правила игры
Расстановка кораблей

Игроку необходимо разместить на поле 10×10:

   1 четырёхпалубный корабль (4 клетки)

   2 трёхпалубных корабля (3 клетки)

   3 двухпалубных корабля (2 клетки)

   4 однопалубных корабля (1 клетка)

Формат ввода: А1г или А10в

   Буква: А, Б, В, Г, Д, Е, Ж, З, И, К

   Цифра: 1-10

   Ориентация: г (горизонтально) или в (вертикально)

Ход игры

   Игрок вводит координаты выстрела (например: А5 или А10)
   При попадании игрок получает дополнительный ход
   Компьютер ходит после промаха игрока
   Игра продолжается до полного уничтожения флота противника

Обозначения на поле
Символ	Значение
~	Вода (не стреляли)

□	Корабль (не повреждён)

X	Попадание

█	Уничтоженный корабль

•	Промах
